# Executive Sales BI Dashboard

[![Python](https://img.shields.io/badge/Python-3.11-blue)](https://www.python.org/)
[![Pandas](https://img.shields.io/badge/Pandas-1.5-green)](https://pandas.pydata.org/)

This project is a **Business Intelligence / Data Engineering** example designed to showcase skills in:

- Data analysis and transformation  
- Executive KPI calculation  
- Data visualization for strategic decision making  

---

## Project Structure

- `data/` : raw datasets and CSV files  
- `notebooks/` : Python notebooks with KPI calculations and analysis  
- `reports/` : Executive PDF reports with KPIs and charts  
- `images/` : Screenshots and visualizations  

---

## Key Content

- **Dataset**: [sales_data.csv](data/sales_data.csv)  
- **Notebook**: [sales_bi_project.ipynb](notebooks/sales_bi_project.ipynb)  
- **Executive Report**: [executive_sales_report_with_charts.pdf](reports/executive_sales_report_with_charts.pdf)  

---

## Purpose

This project demonstrates the ability to **transform raw data into actionable business insights**, ready to be presented to a client or an executive.  

---

## Highlights

- Clear project structure, professional workflow  
- KPI metrics with executive interpretation  
- Charts and visual dashboards integrated into reports  
- Fully reproducible using Python, Pandas, and Matplotlib
